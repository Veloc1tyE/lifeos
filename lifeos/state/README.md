# State Directory

This directory contains current, factual state only.

No philosophy. No plans. No narratives.

These files change daily and are not versioned (except structure).

## Files

| File | Purpose |
|------|---------|
| `STATE.md` | Human-readable snapshot of current state |
| `current-week.json` | Canonical machine state + session handoff |
| `dashboard-live.json` | Live UI feed from browser dashboard |
| `commitments.json` | Active commitments registry |
| `patterns.json` | Longitudinal pattern observations |
| `action-packets.json` | Pre-composed action sequences |
| `inbox.md` | Temporary notes, cleared after addressed |
| `session-log.md` | Session history for continuity |
| `state-queue.json` | Pending state updates from automated sessions |
