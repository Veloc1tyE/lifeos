# LifeOS Cron Configuration
#
# To install: crontab -e and paste these lines
# To view current: crontab -l
#
# Note: These assume Abu Dhabi timezone (GMT+4)
# Adjust times based on your actual wake/sleep schedule
#
# The cron jobs open Terminal and run the triggers.
# You may prefer manual triggers or macOS Shortcuts instead.

# ============================================
# DAILY TRIGGERS
# ============================================

# Morning Calibration - 6:00 AM Abu Dhabi
# (Adjust to your actual wake time)
# 0 6 * * * cd ~/Documents/dashboard && ./lifeos/triggers/morning.sh

# Evening Shutdown - 9:00 PM Abu Dhabi
# (Adjust to your actual shutdown time)
# 0 21 * * * cd ~/Documents/dashboard && ./lifeos/triggers/evening.sh

# ============================================
# WEEKLY TRIGGERS
# ============================================
